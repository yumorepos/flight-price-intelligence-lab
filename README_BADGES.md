## Badge Explanations:

### 1. Tests Badge (GitHub Actions)
- Shows build status (passing/failing)
- Updates automatically on each push
- Links to test workflow runs

### 2. Deploy Badge (GitHub Actions)
- Shows deployment status
- Updates on each deployment
- Links to deployment workflow

### 3. Codecov Badge (Optional)
- Shows code coverage percentage
- Requires Codecov account (free for public repos)
- Setup: https://about.codecov.io/

### 4. License Badge
- Shows MIT license
- Standard for open source projects

### 5. Python Version Badge
- Shows supported Python versions
- Indicates modern tooling

### 6. Next.js Version Badge
- Shows frontend framework
- Technical stack visibility

## How to Add:

1. Open `README.md`

3. Replace `USERNAME` with your GitHub username
4. Replace `REPO_NAME` with `flight-price-intelligence-lab`
5. Commit and push
